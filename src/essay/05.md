#### 什么是跨端

为什么做跨端：学习一种语言，再多端编写，写一套代码，在多端应用

Web 是一种天然的跨端语言，且是通用的

传统 Web 跨端劣势：

- 首屏加载时间长，交互体验差
- 开放生态的环境下，难以管控
